安装使用内网穿透nps的注意事项

Updated on Feb 23, 2020 in 教程 with 9 views and 0 comments
# 开源 # Internet # 教程 # 踩坑之路

官方文档地址：https://ehang-io.github.io/nps/#/?id=nps

官方下载地址：https://github.com/ehang-io/nps/releases

本教程版本v0.26.3

按照官方文档安装：

  1. 按照自己的系统下载 releases，解压出来（注意文件权限）
  2. 把服务端启动在服务器上（有公网 IP 的服务器或外网可以访问的服务器）
  3. 启动好服务端，打开 Web 控制台 http://192.168.56.3:8080/ (ip 换成自己的服务器地址)，登录（默认的账号： admin 密码： 123）
  4. 启动好客户端，启动好后 Web 控制台会出现


    上图启动好了不能退出，已退出就会退出客户端，可以使用服务的方式或使用后台运行的方式去执行

服务启动参考官方文档：

 https://ehang-io.github.io/nps/#/use?id=注册到系统服务

后台启动：

./npc -server=192.168.56.3:8024 -vkey=123456 -type=tcp &


5. 这时需要穿透内网8080端口到外网88端口有点坑，我使用http好像没法达到这个效果，所以我使用的tcp隧道


6. 如上图配置完成后就能通过外网浏览器访问内网链接了
